# 물냥이 Water_Cat
## 개요
- 우리집 어떤 고양이가 얼만큼 물을 마시는지 측정한다. 

## 기술 
- openCV 이미지인식 
- 카메라 
- 로드셀(무게센서)
- wifi 


## 부품 및 제원 
- 라즈베리파이 4 
- 4선식 로드셀 2kg
- 로드셀 앰프 24비트 A/D 컨버터 HX-711
- 


### 부품 조립
1. 아크릴 판 / 나무판 2개 구해서 로드셀 장착 
2. 로드셀 - HX711 컨버터 모듈 - 라즈베리파이 연결 
3. 카메라 모듈 - 라즈베리파이 연결 

### 연결 및 조작 
1. 2.4gb 와이파이 - ssh연결 
2. 포트 열어서 터미널 접속 
3. 이후 애플워치 앱 / 핸드폰 앱 개발
